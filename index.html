// how it works

Three steps.
One smooth result.

AniScript splits the problem into three clean layers — you only touch the first one.

01
Write DSL
Describe your animation in a concise, readable syntax. No CSS classes to memorize, no keyframes to write.
:: fade-up | @200ms :: {
  Your content here
}
02
Compile to HTML
The compiler transforms your DSL into semantic HTML with data attributes. Works at build time or runtime.
<div
  data-ani="fade-up"
  data-ani-delay="200ms"
>...</div>
03
Runtime Triggers
The IntersectionObserver watches each element and fires the CSS animation the moment it enters the viewport.
init({ threshold: 0.1 });

// Fires as user scrolls

// get started

Up in 60 seconds.

Two paths — pick what fits your project.

NPM / Build Tool
For React, Vue, Svelte, Next.js, or any modern build pipeline.
# 1. Install
npm install aniscript

import { compile, init } from 'aniscript';
import 'aniscript/dist/aniscript.css';

// 2. Compile & inject
const html = compile(':: fade-up :: { Hello }');
document.querySelector('#app').innerHTML = html;

// 3. Start the observer
init();
CDN / Static Site
Drop into any HTML file. No build step required.
<!-- In <head> -->
<link rel="stylesheet"
 href="https://unpkg.com/aniscript/dist/aniscript.css">

<!-- Before </body> -->
<script type="module">
 import { compile, init }
  from 'https://unpkg.com/aniscript/src/index.js';
 document.body.innerHTML +=
  compile(':: bounce-in :: { Hello! }');
 init();
</script>

Simple API.
Powerful results.

The full DSL parameter reference at a glance.

ParameterTypeDefaultDescription
animation-namestringrequiredAny of the 53 built-in names. e.g. fade-up, zoom-in, bounce-left, shake-h
tagdiv | span"div"The HTML wrapper element. Use span for inline content.
@delaytime"0ms"Delay before animation starts. e.g. @200ms, @0.5s
dur:durationtime"1s"Total animation duration. e.g. dur:800ms, dur:1.5s
[[ stagger: t ]]time—Wraps children and applies cumulative delays. Close with [[/]]
init(config)object{}threshold (0–1) and rootMargin for the IntersectionObserver.
